# About This Repository

## Background

This is a repository made by etoile_et_toi.mp3 during the second semester of the 114 academic year (Feb. 2026 ~ Jun. 2026). This repository is primarily dedicated to the "Introduction to Image Processing" course, which is instructed by A.P. Hsu-Feng Hsiao (蕭旭峰) at National Yang Ming Chiao Tung University, Taiwan.

## Purpose

The purpose of this repository is to document the author's learning progress and to serve as a resource for anyone interested in the course material. By sharing this content openly, the repository aims to inspire discussions and spark new ideas related to the course's topics and tasks.

## Content

The content in this repository includes the slides provided by the instructor and the source code created by the author for the homework and labs given in this course.
